flaskr.create_app()[source]

Create and configure an instance of the Flask application.

Enables CORS (Cross-Origin Resource Sharing) to allow requests from web applications hosted on different domains. Registers the article_route blueprint to manage article-related routes.

Returns:

Flask app: The configured Flask application instance.
